## ✨ Features
* **Job Tracker:** Full lifecycle management (Applied, Interview, Offer, Rejected).
* **Interview Manager:** Reminders and scheduling with Alarm Manager.
* **CV Hub:** Secure storage for resumes via Firebase Storage.
* **Smart Analytics:** Visual progress tracking using Vico charts.
