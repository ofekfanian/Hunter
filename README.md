## 💡 The Concept: The Junior's Edge

While competitors exist, they are often built for English-speaking markets, require subscription fees for top features, or focus on desktop-first experiences. **Hunter** is built specifically for the **Israeli Student & Junior Developer** entering the tech force:

* **⚡ Speed Apply:** Zero-friction application logging. Found a job on LinkedIn? Use "Quick Add" to save a screenshot and link instantly.
* **📂 CV Versioning (The Killer Feature):** Tracks exactly which CV version (`Fullstack`, `Backend`, `Data`) you sent to which company.
* **🤝 Interview Intelligence:** A social hub where the community shares real interview questions from specific companies (e.g., "What did Wix ask in their technical round?").

## 🛠️ Technical Implementation

### **UI/UX Strategy**
* **Color-Coded Architecture:** Each application area has a dedicated color (e.g., `Indigo` for Jobs, `Rose` for Social), making navigation intuitive.
* **Motion Design:** Uses custom XML animations for "fall down" list effects and smooth Activity transitions.
* **Material 3:** Leveraging the latest components for a modern, premium dark theme.

### **Backend & Services**
* **Real-time sync:** Firebase Firestore keeps your application pipeline synchronized across all devices.
* **Cloud Storage:** Secured CV PDFs and job screenshots hosted on Firebase Storage.
* **Reliability:** AlarmManager integration ensures you never miss a scheduled interview.
